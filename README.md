### 2.1 Image binarization processing and grayscale level processing

1. Maximum gray processing method
   $$
   \text{gray}(i,j)=\max\left(R(i,j),G(i,j),B(i,j)\right)
   $$

2. Average gray processing method
   $$
   \text{gray}(i,j)=\frac{R(i,j)+G(i,j)+B(i,j)}{3}
   $$

3. Weighted average gray processing method
   $$
   \text{gray}(i,j)=0.30\times R(i,j)+0.59\times G(i,j)+0.11\times B(i,j)
   $$
