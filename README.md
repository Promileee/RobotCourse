# Challenging Practice for Intelligent Robot

This repository stores documents and files developed during the Challenging Practice for Intelligent Robot course. Cooperator: [Jiang Weiyang](https://github.com/zzzpenelo).

## Experiments

### Experiment 1 — Color Channels

Basic image and video processing using OpenCV. Covers real-time camera capture with channel splitting (R/G/B) and grayscale conversion, as well as static image channel visualization with matplotlib.

Files: [camera1.py](experiment1/camera1.py), [picture1.py](experiment1/picture1.py), [photo1.png](experiment1/photo1.png)

### Experiment 2 — Image Enhancement

Image enhancement techniques including:

- **Binarization** — manual thresholding (64, 128, 192) and OTSU's method
- **Gray-level quantization** — reducing gray levels to 4, 8, 16, 32
- **Gamma correction** and **log transform** — brightness/contrast adjustment
- **Histogram equalization** and **CLAHE** — contrast enhancement
- Real-time camera dashboard and static image processing with result export

Files: [camera2.py](experiment2/camera2.py), [picture2.py](experiment2/picture2.py), [process2.py](experiment2/process2.py), [results.pdf](experiment2/results.pdf)

### Experiment 3 — Advanced Image Processing

A comprehensive Jupyter notebook covering:

- **Image filtering** — custom kernel convolution, mean/median/Gaussian filtering for denoising
- **Laplacian sharpening** — image detail enhancement
- **Morphological operations** — erosion, dilation, opening, closing on fingerprint images
- **Fourier transform** — frequency-domain analysis and low-pass filtering
- **Edge detection** — Sobel and Canny operators
- **Hough transform** — line and circle detection

Files: [experiment3.ipynb](experiment3/experiment3.ipynb)
